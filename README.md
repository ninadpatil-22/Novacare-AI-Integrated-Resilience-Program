# NovaCare AI — Integrated Resilience Program

A complete GRC portfolio project simulating a full organizational 
resilience program for a fictional healthtech SaaS company processing 
PHI and cardholder data.

## Company Profile
NovaCare AI — 320-employee healthtech SaaS startup based in Austin, TX. 
Processes PHI (HIPAA), cardholder data (PCI DSS), and PII (GDPR/CCPA). 
Infrastructure hosted on AWS (us-east-1 primary, us-west-2 DR).

## Scenario
A ransomware attack encrypts production systems and exfiltrates patient 
data, testing the organization's full resilience program — from impact 
analysis through crisis response, recovery, and lessons learned.

## Frameworks Applied
ISO 22301:2019 | NIST CSF | ISO 27001:2022 | HIPAA | PCI DSS v4.0

## Deliverables

| # | Document | Description |
|---|---|---|
| 01 | Business Impact Analysis | Critical process register, RTO/RPO, impact scoring, dependency mapping |
| 02 | Crisis Management Plan | CMT roster, decision authority, crisis declaration thresholds |
| 03 | Disaster Recovery Plan | System recovery sequence, runbooks, DR test schedule |
| 04 | Ransomware IRP + Decision Tree | 6-phase incident response lifecycle with visual decision tree |
| 05 | Business Continuity Plan | 3 disaster scenarios: ransomware, cloud outage, key personnel loss |
| 06 | Tabletop Exercise Script | 3-inject ransomware simulation for CMT training |
| 07 | Framework Control Matrix | Maps all deliverables against 5 compliance frameworks |
| 08 | Post-Incident Review | PIR template + worked example from tabletop scenario |
| 09 | ServiceNow Screenshots | IR workflow and BCP task logging in ServiceNow |
